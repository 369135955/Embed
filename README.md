## 知识学堂
支付宝即时到账接口解决XP、2003系统无法打开支付宝网站
由于支付宝免签约即时到帐辅助内调用IE内核访问支付宝网站页面，而支付宝网站的ssl证书不再支持XP、2003系统，会导致XP、2003系统在使用IE浏览器访问支付宝网站时出现找不到服务器的情况。
